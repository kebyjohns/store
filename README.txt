kebymage — Multi‑page site
================================

Files
-----
- index.html
- photography.html
- videography.html
- web-design.html
- graphic-design.html
- store.html
- styles.css
- app.js

Finish Setup
------------
1) Contact form:
   - Create a Formspree form and replace YOUR_FORMSPREE_ID in index.html

2) PayPal checkout (store.html):
   - Replace YOUR_PAYPAL_CLIENT_ID in the PayPal SDK script URL.
   - Products are demo-only in app.js (20 items incl. Haitian merch).

3) Deploy:
   - Upload all files to your host or drag the folder into Netlify/Vercel.
